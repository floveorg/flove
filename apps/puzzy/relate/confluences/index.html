My Favourite ConfluencesMis Confluencias Favoritas

Lists that converge — each one centred on its confluence.Listas que convergen — cada una centrada en su confluencia.

How to use: each row is one list. Add a class len-1 … len-7 matching how many items it has, and place items in natural order (first → last). The grid spreads them across the 7 columns, centred. Special rule for len-2: item 1 → column 1, item 2 → column 7. Cómo usar: cada fila es una lista. Añade una clase len-1 … len-7 según cuántos elementos tenga, y coloca los elementos en orden natural (primero → último). La rejilla los reparte por las 7 columnas, centrados. Regla especial para len-2: elemento 1 → columna 1, elemento 2 → columna 7.
